Bfthon 🧠⇄🐍

A lightweight Python module for bidirectional conversion between Pythton strings and Brainfuck code.

Source Code

Features

  • 🔄 Bi-directional Conversion
  • 🚀 Zero Dependencies
  • 🐍 Python 3.6+
  • 📦 Lightweight

    • Usage Examples

    from bfthon import encode, decode

# Encoding
bf_code = bfthon.encode('print("Hello World")')
print(bf_code)
# Output: ++++++++[>++++[...]

# Decoding
output = bfthon.decode("++++++++[>++++[...]")
print(output)
# Output: Hello World
